# 验证模拟器特性支持流程

## 触发条件
用户询问某特性在模拟器中的支持情况。

## 检查步骤
1. 确定特性名称和所属协议层。
2. 分析特性是否依赖硬件能力：
   - 查看厂商文档中的硬件最低要求。
   - 如果文档未提及硬件，通常可以模拟。
3. 验证模拟器中软件版本和配置语法：
   - 使用 `show version` 确认镜像版本。
   - 尝试配置命令，观察是否报错。
4. 验证功能行为：
   - 使用对应的 `show` 命令查看表项。
   - 模拟客户端/服务器交互。
5. 记录限制：性能、吞吐量、特定硬件特性不可模拟。

## 输出格式
返回结论文档，包含可模拟/有限制/不可模拟及原因。
